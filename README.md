# CTS Digital Nurture Program

This repository contains the learning materials, hands-on exercises, and practice programs completed as part of the **Cognizant Digital Nurture Program (DNP)**.

## Repository Structure

```

├── Upskilling

│   ├── CSS3
```

## Contents

### Upskilling
The Upskilling section covers fundamental concepts required for full-stack development.

Topics include:
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- jQuery
- SQL
- ANSI SQL using MySQL
- C# ADO.NET

### DeepSkilling
The DeepSkilling section focuses on advanced backend development concepts and enterprise application development.

Topics include:
- Java
- Spring Framework
- Spring Boot
- REST APIs
- Microservices
- Additional advanced modules

## Purpose

This repository is maintained to:

- Maintain all learning resources in one place
- Track progress throughout the training
